# toolab-core

Go runtime and CLI for Toolab.

Implementation follows phased delivery documented at repository root.

## Commands

- `toolab run <scenario.yaml> [--out DIR]`
- `toolab generate --from openapi|toolab ...`
- `toolab enrich <scenario.yaml> --from openapi|toolab ...`
- `toolab map --from openapi|toolab ...`
- `toolab explain <run_dir> ...`
- `toolab diff <runA_dir> <runB_dir> ...`
