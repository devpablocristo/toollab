# toollab-core

Go runtime and CLI for Toollab.

Implementation follows phased delivery documented at repository root.

## Commands

- `toollab run <scenario.yaml> [--out DIR]`
- `toollab generate --from openapi|toollab ...`
- `toollab enrich <scenario.yaml> --from openapi|toollab ...`
- `toollab map --from openapi|toollab ...`
- `toollab explain <run_dir> ...`
- `toollab diff <runA_dir> <runB_dir> ...`
